一、核心知識拆解

矩陣與像素影像就是一個數值矩陣，每個格子是像素。
img.shape → (H, W, 3)｜dtype → uint8｜值域 0~255
影像讀取與通道順序cv2.imread 回傳 BGR（不是 RGB）；PIL.Image.open 回傳 RGB。混用時務必轉換。
色彩空間轉換同一個顏色改用不同座標系描述。RGB 看混色、Gray 看亮度、HSV 看色相。灰階公式：0.299R + 0.587G + 0.114B（G 權重最高因人眼最敏感）。
幾何轉換改變影像的位置、角度或形狀。旋轉用 getRotationMatrix2D + warpAffine；翻轉用 cv2.flip（0=垂直,1=水平,-1=兩者）。
卷積用小視窗掃過整張圖，重新計算局部。模糊、銳化、邊緣偵測都是卷積的應用，差別在 kernel 值不同。
降噪與平滑降低影像中的隨機雜訊。Gaussian Blur 最常用，先降噪再找邊緣通常更穩定。
邊緣偵測找亮度變化劇烈的位置。Canny 是最重要的方法，需設定雙閾值（低閾值排弱邊緣、高閾值保強邊緣）。
二值化與閾值方法手動（THRESH_BINARY）、Otsu 自動閾值、自適應閾值（每個區域獨立計算）。考試常考差異。
直方圖與均衡化直方圖顯示像素亮度分布。均衡化（equalizeHist）重新分配使分布均勻，用於提升低對比影像的對比度。
形態學操作膨脹(dilate) → 侵蝕(erode)，組合成開運算（去小白點）、閉運算（補小黑洞）。記憶：開 = 侵蝕→膨脹，閉 = 膨脹→侵蝕。
插值方法縮放時填補新像素的策略。INTER_NEAREST（最快，馬賽克感強）、INTER_LINEAR（雙線性，最常用）、INTER_CUBIC（最平滑）。
CNN 前處理流程讀取 → resize 到固定尺寸 → 轉 RGB → 除以 255 正規化到 [0,1] → 轉 tensor。這是 AI pipeline 的標準起點。

實驗室 B：卷積 / 濾波 / 邊緣 / Canny

Canny 需設定雙閾值（低閾值＋高閾值），這是 iPAS 高頻考點。
記憶：Canny = 先降噪，再雙閾值切強弱邊緣。

建議觀察：模糊 → 細節變淡；銳化 → 對比提高；Laplacian → 所有邊界；Canny → 只保留強邊界。 
import cv2
import numpy as np

img = cv2.imread("test.jpg", 0)  # 0 = 灰階

# 高斯模糊

# 銳化 (自定義 kernel)
kernel = np.array([[0,-1,0],[-1,5,-1],[0,-1,0]])
sharp = cv2.filter2D(img, -1, kernel)

# Laplacian 邊緣
lap = cv2.Laplacian(img, cv2.CV_64F)
lap_abs = np.uint8(np.absolute(lap))

# Canny 邊緣 ★ 考試重點 ★
# 先 Gaussian 降噪，再雙閾值
edges = cv2.Canny(img,
    threshold1=50,   # 低閾值：弱邊緣門檻
    threshold2=150)  # 高閾值：強邊緣門檻

實驗室 E：直方圖與均衡化 ★ 考試重點 ★

直方圖顯示各亮度值（0~255）的像素數量分布。
均衡化（equalizeHist）將分布拉平，使暗區也有豐富細節。
觀察重點：均衡化後直方圖分布更均勻，影像對比更強。
記憶：equalizeHist = 把餅乾切平，讓每個亮度都有份。 

import cv2
import numpy as np
import matplotlib.pyplot as plt

img = cv2.imread("test.jpg", 0)  # 灰階

# 計算直方圖
hist = cv2.calcHist([img], [0], None, [256], [0, 256])

# 直方圖均衡化 ★
equalized = cv2.equalizeHist(img)
hist_eq = cv2.calcHist([equalized], [0], None, [256], [0, 256])

# 可視化對比
plt.subplot(2,2,1); plt.imshow(img, cmap="gray"); plt.title("原圖")
plt.subplot(2,2,2); plt.plot(hist); plt.title("原始直方圖")
plt.subplot(2,2,3); plt.imshow(equalized, cmap="gray"); plt.title("均衡化")
plt.subplot(2,2,4); plt.plot(hist_eq); plt.title("均衡化直方圖")
plt.tight_layout(); plt.show()

實驗室 F：閾值方法比較 ★ 考試重點 ★

Otsu 自動閾值：自動找出使前景與背景變異最大的最佳閾值。不需要手動調整，適合亮度分布雙峰的影像。
cv2.threshold(img, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)
自適應閾值：對影像中每個區域獨立計算閾值，適合光源不均勻的影像。
cv2.adaptiveThreshold(img, 255, cv2.ADAPTIVE_THRESH_GAUSSIAN_C, cv2.THRESH_BINARY, 11, 2)

三者差異記憶：
Binary = 自己設門檻｜Otsu = 自動找最佳門檻｜Adaptive = 每個區塊各自設門檻

import cv2

img = cv2.imread("test.jpg", 0)

# 1. 手動 Binary ★
_, binary = cv2.threshold(
    img, 127, 255, cv2.THRESH_BINARY)

# 2. Otsu 自動閾值 ★★
# threshold 參數設 0，讓 Otsu 自己算
_, otsu = cv2.threshold(
    img, 0, 255,
    cv2.THRESH_BINARY + cv2.THRESH_OTSU)

# 3. 自適應閾值 ★★
# blockSize=11 (每格大小), C=2 (修正常數)
adaptive = cv2.adaptiveThreshold(
    img, 255,
    cv2.ADAPTIVE_THRESH_GAUSSIAN_C,
    cv2.THRESH_BINARY, 11, 2)

    interpolation=cv2.INTER_LINEAR)

四、關鍵理解強化

常見迷思 ★

cv2.imread = BGR，不是 RGB
使用 matplotlib 顯示 cv2 讀取的圖要先 cvtColor！

img.shape = (H, W, C)
不是 (W, H, C)！H=rows=height，W=cols=width。

PIL.size = (W, H)
PIL 的 size 屬性順序和 shape 相反，要小心。

灰階 ≠ 二值圖
灰階有 256 個亮度層；二值化才是非黑即白。

開運算 ≠ 先膨脹
開運算 = 先侵蝕後膨脹；閉運算 = 先膨脹後侵蝕。

CNN 前處理要點 ★

為什麼除以 255？
將像素從 [0,255] 壓縮到 [0.0,1.0]，有助於梯度穩定、加速收斂。

為什麼先轉 RGB？
大多數預訓練模型（ResNet、VGG）以 RGB 圖訓練，直接用 cv2 的 BGR 會導致色彩錯誤。

HWC vs CHW：
NumPy 預設 HWC；PyTorch 需要 CHW；TensorFlow 通常接受 HWC。
